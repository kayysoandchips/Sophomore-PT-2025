Networking Overview

networking componets

  • Router: Connects different networks and directs data between them
  • Switch: Connects multiple devices within the same network to enable communication
  • Modem: Converts internet signals from your ISP for use in your network
  • Access Point (AP): Expands wireless coverage in a network
  • Firewall: Monitors and controls incoming/outgoing network traffic for security
  • Network Interface Card (NIC): Hardware that connects a computer to a network
  • Ethernet Cable: Physical medium for wired connections between devices
  • Server: Provides services like file storage, hosting, or apps to client devices
  • Client Devices: Devices like computers, phones, or tablets that access network resources
  • Repeater/Extender: Boosts or relays signals to extend network coverage
  • Gateway: Acts as a bridge between different systems or networks
  • Patch Panel: Organizes and connects multiple network cables in larger setups
