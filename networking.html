Networking Overview

Networking Components

  • Router: Connects different networks and directs data
  • Switch: Links devices within the same network
  • Modem: Converts signals from your ISP
  • Access Point (AP): Expands wireless coverage
  • Firewall: Controls network security
  • NIC: Connects a computer to a network
  • Ethernet Cable: Wired connections between devices
  • Server: Offers data/services to other devices
  • Client Devices: Computers, phones, etc.
  • Repeater: Boosts signal range
  • Gateway: Bridges different networks
  • Patch Panel: Organizes network cables

Network Addressing

  • IP Address: Unique ID for devices
  • MAC Address: Hardware address for network interfaces
  • Subnet: Network segmentation
  • Gateway: Connects to external networks
  • DNS: Translates web names into IPs

Benefits of Networking

  • Faster communication
  • Shared hardware and software
  • Central data management
  • Cost-effective for businesses
  • Supports remote access

Types of Networks

  • LAN: Local network (home/school)
  • WAN: Wide-area network
  • MAN: City-wide coverage
  • WLAN: Wireless LAN
  • PAN: Personal area network
